# priflix ⚛️

![priflix logo](./src/assets/img/Logo.png)

Projeto desenvolvido durante as aulas de ** imersão de React ** pela Alura

## Como instalar

Baixe o projeto:

```
git clone https://github.com/PriscilaNetto/priflix.git

```

## e no terminal digite os seguintes comandos:

```
- npm install
- npm start
- npm run dev
```
